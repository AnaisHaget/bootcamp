Exercise 2.2: The sieve of Eratosthenes¶

The sieve of Eratosthenes is a method for computing prime numbers. Read about it on Wikipedia. Based on the pseudocode, write a Python code to make a list of all primes less than 10,000. How many are there? (When you answer this question, you’ll see the curious result that the number of primes less than 10,000 is also prime!)

Solution¶

We have not yet learned how to use Python for computing square roots, but so we will hand-enter the values.

[1]:

n = 10000
sqrt_n = 100

Now, we can code up the sieve.

[2]:

# Array of Trues and Falses for whether or not a number is prime
isprime = [False]*2  + [True] * (n - 1)

# Look through possible prime factors
for i in range(2, sqrt_n):
    if isprime[i]:
        j = i**2
        while j <= n:
            isprime[j] = False
            j += i

# Based on Trues in the sieve, make a list of primes
primes = []
for i, prime in enumerate(isprime):
    if prime:
        primes.append(i)
